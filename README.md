# Lettera

<img src="public/lettera.png" width="100" height="auto" alt="Description">

A desktop plain-text and Markdown editor built with Tauri 2, React, and
TypeScript.

## Requirements

- Node.js
- pnpm
- Rust
- Platform-specific dependencies required by Tauri 2

## Development

```shell
pnpm install
pnpm tauri dev
```

## Build

```shell
pnpm tauri build
```
